django

django is a server scripting language, and a powerful tool for making dynamic and interactive Web pages.

django is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP.

django is perfectly suited for Web development and can be embedded directly into the HTML code.

django is often used together with Apache (web server) on various operating systems.

django is easy to learn and runs efficiently on the server side.

django is open source software.

django is free to download and use.
